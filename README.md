# spamc


## Python spamassassin spamc client library

spamc is a python module that provides all the functionality of the spamc client
program.

## Installation

Install from PyPi

    pip install spamc

Install from Githib

    git clone https://github.com/akissa/spamc.git
    cd spamc
    python setup.py install
